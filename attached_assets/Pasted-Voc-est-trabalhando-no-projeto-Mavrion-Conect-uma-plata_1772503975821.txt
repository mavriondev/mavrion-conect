Você está trabalhando no projeto Mavrion Conect, uma plataforma B2B em TypeScript/Node.js com Express, PostgreSQL e Drizzle ORM.

TAREFA: Implementar um sistema de cache em banco de dados para as chamadas às APIs externas ANM, SICAR e CNPJA. O cache deve usar a tabela `raw_ingests` que já existe no schema em `shared/schema.ts`.

A tabela `raw_ingests` tem os campos:
- `connectorId` (integer, opcional — pode ser null para cache manual)
- `externalId` (text) — use como chave do cache (ex: "anm:processos:MG:OURO")
- `payloadJson` (jsonb) — o resultado da API
- `hashDedupe` (text) — use como TTL encoded (ex: "ttl:2025-12-31T10:00:00Z")
- `fetchedAt` (timestamp) — data da busca

IMPLEMENTAÇÃO:

**1. Criar utilitário de cache em `server/cache.ts`:**
```typescript
import { db } from "./db";
import { rawIngests } from "@shared/schema";
import { eq } from "drizzle-orm";

const TTL_SECONDS = {
  anm: 60 * 60 * 6,        // 6 horas — dados minerários mudam pouco
  sicar: 60 * 60 * 12,     // 12 horas — CAR é quase estático
  cnpja: 60 * 60 * 24,     // 24 horas — dados da Receita mudam raramente
  geo: 60 * 60 * 24,       // 24 horas — hidrografia/energia/elevação são estáticos
};

type CacheNS = keyof typeof TTL_SECONDS;

export async function getCached<T>(namespace: CacheNS, key: string): Promise<T | null> {
  const cacheKey = `${namespace}:${key}`;
  const [row] = await db.select().from(rawIngests).where(eq(rawIngests.externalId, cacheKey));
  if (!row) return null;

  // hashDedupe armazena a data de expiração
  const expiresAt = new Date(row.hashDedupe);
  if (new Date() > expiresAt) {
    // Cache expirado — delete para não acumular lixo
    await db.delete(rawIngests).where(eq(rawIngests.externalId, cacheKey));
    return null;
  }

  return row.payloadJson as T;
}

export async function setCached<T>(namespace: CacheNS, key: string, data: T): Promise<void> {
  const cacheKey = `${namespace}:${key}`;
  const ttl = TTL_SECONDS[namespace];
  const expiresAt = new Date(Date.now() + ttl * 1000).toISOString();

  // Upsert: deleta o anterior se existir, insere novo
  await db.delete(rawIngests).where(eq(rawIngests.externalId, cacheKey));
  await db.insert(rawIngests).values({
    externalId: cacheKey,
    payloadJson: data as any,
    hashDedupe: expiresAt,  // reutilizamos hashDedupe para guardar TTL
    fetchedAt: new Date(),
  });
}
```

**4. Adicionar endpoint para limpar cache manualmente em `server/routes/system.ts`:**
```typescript
app.delete("/api/cache/flush/:namespace", async (req, res) => {
  if (!req.isAuthenticated()) return res.status(401).json({ message: "Unauthorized" });
  const user = req.user as any;
  if (user.role !== "admin") return res.status(403).json({ message: "Forbidden" });
  
  const ns = req.params.namespace; // "anm", "sicar", "cnpja", "geo", ou "all"
  if (ns === "all") {
    await db.delete(rawIngests);
  } else {
    // Deletar todos os registros cujo externalId começa com o namespace
    await db.execute(sql`DELETE FROM raw_ingests WHERE external_id LIKE ${ns + ":%"}`);
  }
  res.json({ success: true, message: `Cache ${ns} limpo.` });
});
```
