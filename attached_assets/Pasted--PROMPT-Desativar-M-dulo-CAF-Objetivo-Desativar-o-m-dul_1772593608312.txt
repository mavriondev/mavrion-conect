# PROMPT — Desativar Módulo CAF

## Objetivo
Desativar o módulo CAF do sistema sem apagar dados do banco.
Apenas remover rotas, esconder menus e bloquear navegação.

**Não apagar nada do banco de dados. Não apagar arquivos de código.**

---

## ETAPA 1 — Remover as rotas CAF do App.tsx

Abra o arquivo: `client/src/App.tsx`

Localize e **apague** os dois blocos de rota abaixo:

```tsx
<Route path="/caf">
  {() => <PrivateRoute component={NorionCafPage} />}
</Route>
```

```tsx
<Route path="/norion-app/caf">
  {() => <NorionPrivateRoute component={NorionCafPage} />}
</Route>
```

Mantenha o import do `NorionCafPage` por ora — só remova as rotas.

Salve o arquivo.

---

## ETAPA 2 — Remover o item CAF dos menus laterais

### 2.1 Sidebar do Mavrion

Abra o arquivo do sidebar principal do Mavrion.
Procure por qualquer item de menu que contenha o texto `"CAF"` ou o link `"/caf"`.
**Comente ou apague** esse item do menu.

### 2.2 Sidebar do Norion

Abra o arquivo do sidebar do Norion (`components/norion-layout.tsx` ou similar).
Procure por qualquer item de menu que contenha o texto `"CAF"` ou o link `"/norion-app/caf"`.
**Comente ou apague** esse item do menu.

Salve os arquivos.

---

## ETAPA 3 — Remover o registro das rotas do servidor

Abra o arquivo que registra todas as rotas do servidor
(o arquivo onde existem chamadas como `registerNorionRoutes(app)`).

Se existir uma linha como:
```typescript
registerCafExtratorRoutes(app);
```
**Comente ou apague** essa linha.

Se existir um import como:
```typescript
import { registerCafExtratorRoutes } from "./routes/caf-extrator";
```
**Comente ou apague** esse import também.

Salve o arquivo.

---

## ETAPA 4 — Verificar se funcionou

Reinicie o servidor:
```bash
npm run dev
```

Confirme que:
- O item CAF não aparece em nenhum menu lateral
- Acessar `/caf` ou `/norion-app/caf` no navegador redireciona para 404 ou página inicial
- O restante do sistema continua funcionando normalmente
